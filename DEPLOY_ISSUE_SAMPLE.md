# デプロイ issue サンプル

GitHub 上で以下の内容で issue を作成してください。
作成後に `claude-fix` ラベルを貼ると自動で処理が始まります。

---

## タイトル

```
[fix] startボタンを押しても始まらない
```

## 本文

```
index.html の画面中央にあるstartボタンを押すとゲームが開始されるように修正してください．

### 完了条件
- index.thml startボタン押下時ゲームが始まる
```

## ラベル

`claude-fix`

---

issue を作成した後，`claude-fix` ラベルを貼るだけで Actions が自動起動します。
